อันนี้โพสขายของ ต้องเล่าก่อนว่าผมมีโครงการส่วนตัวอยู่ 2-3 อย่างที่อยากทำมาหลายปีละแต่ก็หยิบๆวางๆมาตลอด ปีที่แล้วก็ใกล้จะได้ทำแต่ไปไม่ถึงไหน มาเริ่มสำเร็จเอาปีนี้แหละด้วยพลังของ chatGPT ที่อยากรู้อะไร สงสัยแบบไหนมันก็ช่วยรวบรวมให้แบบไม่เสียเวลาไปถามแบบโง่ๆตามฟอรั่มเฉพาะทาง

---
เกริ่นมาอ้อมหน่อยว่าเป็นแผนการหลายปีแล้ว  เพราะมีเหตุให้หมดรมณ์จากหลายๆเรื่องก็เลยมีเวลาว่างไปทำ ไม่ได้เพราะมีปัญหาก็เลย 'แยกตัว' มาทำนะ ผมยังอยู่ดีแค่ยังไม่มีรมณ์ สองอย่างนี้มันเขียนดูคล้ายๆแต่บริบทและวิธีคิดต่างกันมาก

---
คือผมอยากทำเวบไว้เก็บบทความตัวเองมานานละ เพราะพอทำนานๆแล้วหาเจอยากแต่ก็มีไอเดียในหัวหลายท่ามาก เมื่อก่อนทำ wordpress แต่พื้นเต็มแล้วไม่อยากจ่ายเพิ่ม อยากลองท่าอื่นด้วย (ด้วยความที่เป็นสายเทคก็อยากลองใช้เทคโนโลยีหลายๆแบบเป็นการเรียนรู้ไปในตัว) ทำๆแล้วเบื่อก็หยุดแบบกลับมาอีกทีอะไรกันว่ะเนี่ยก็บ่อย แต่รอบนี้คิดว่าจริงจังขึ้นเพราะว่าเริ่มสนุกละ

---
ผมมีโครงการอยู่ 3 อย่างครับ เริ่มลงมือแล้วก็เอามาแชร์ก่อนอันหนึ่งก็คือ

1 - เวบบอร์ด - ใช่ platform สโลว์ไลฟ์ที่คนไม่ค่อยใช้กันแล้วนี้แหละ คือผมคิดว่าบทความเนี่ยทำรวมๆไว้ก็อยากให้มันหาง่ายๆด้วย แล้วก็คิดว่าถ้ามีคนสนใจอยากใช้ก็อยากทำตัวเองเป็น micro community แบบมีไว้เก็บบทความกับคำถาม/ตอบดีๆก็อาจจะน่าสนใจ คืออยากคุยไวๆแล้วไหลไปตอนนี้ fb,ig,x,tiktok, line, discord,reddit มันมีอยู่แล้ว อยากย้อนตัวเองกลับไปสมัยจุดเริ่มต้นที่คุยกันในเวบบอร์ดด้วยละ

อยากชวนไปดูและให้คำแนะนำได้ที่ forum.tinymeepletalk.com ครับ อยากทำเป็นที่เปิดที่ใครๆก็ใช้ได้ด้วย ถ้าอยากให้เพิ่ม tag เฉพาะก็มาคุยกันได้

ตอนนี้ก็มีแค่บทความผมเท่านั้นแหละ มีเปิด fb group ไว้ด้วยชื่อเดียวกันแต่ตั้งใจไว้แค่เอาไว้สื่อสารเฉพาะกลุ่มเวลามีข่าวหรือเรื่องอะไรที่อยากคุยเร็วๆมากกว่านะ เพราะไม่งั้นมันก็จะผิดกับไอเดียตั้งต้นว่ามาทำเวบบอร์ดมาทำไม

ซึ่งก็ย้ำอีกทีว่า target group กับ usage รวมไปถีง mood & tone มันต่างกันมาก ผมไม่คิดว่าการไปเปิดกลุ่มเพิ่มมันจะแก้ pain point อะไร เพราะปัญหาคือธรรมชาติของ platform อยู่แล้ว ก็มองว่ามันเป็นอีกพื้นที่ละกัน แบบอยากผมได้สังคมแบบ search ก่อนถาม พอมาแบบไม่รู้ไรเลยก็อยากโยนลิงค์ไปอ่านตรงนี้ก่อนสิฟระ! แบบเข้มๆไม่ต้องเกรงใจใครงี้

2 - เดี๋ยวว่าจะแยกเอา content ไปทำเป็น blog ด้วยก็ไม่ได้แฟนซีอะไร แค่มีทั้งรีวิวกับบ่นที่เรียกว่าก็อป fb ไปลงเฉยๆ (จริงๆคือในมุมเทคอยากลองใช้ astro flamework + deploy ลง netlify)

3 - เคยคุยไว้นานละแต่อยากลองทำพวกเวบเอาไว้ filter แนะนำเกมตามหมวดหมู่หัวเรื่องไรงี้ แล้วก็ทำเป็นการค้าจางๆใส่ลิงค์ affiliate ไปเลย แบบใครถามปุ๊บเอาลิงค์พี่ไปเลือกเกมดูนะไรงี้ (ในทางเทคกะว่าจะลอง flutter web + firebase db )

ซึ่งสองอย่างนี้ยังไม่ได้ทำ แต่เดี๋ยวเสร็จอันนี้ไปทำต่อ

ตัวฟอรั่มผมใช้ Flarum ตอนแรกจะเช่าโฮสในไทยแต่พอดี Flarum มันต้องใช้ SSH เพราะต้องใช้ composer ในการติด package ดูราคาแล้วตอนแรกขี้เกียจจ่ายเลยสำหรับโปรเจคของเล่น ก็เลยไปลองท่า google cloud vm มา แบบโหสร้างฟรีเว้ย ถาม chatGpt setup อย่างดีสรุปติดนิดเดียวมันเรียกเก็บค่า network เลยเอาความรู้ที่ได้มาไปเช่าพื้นที่ใน vultr แทน ไม่แพงมาก 1 CPU 1GB Ram ส่วน hdd ไปใช้ของ cloudflare R2 ได้ฟรี 10GB คชจ. ตอนนี้เลยอยู่ที่เดือนละ 5 USD ซึ่งถ้า server มันรับไหว (เพราะไม่ได้กะว่ามีคนใช้เยอะแยะ) ก็น่าจะสบายๆอยู่นะ ส่วน blog กะ ตัว web app เดี๋ยวไปดูก่อน ตอนแรกอยากใช้ท่า static web แต่ก็แอบกลัวว่า deploy update บ่อยๆแล้วรำคาญตัวเองเหมือนกัน
