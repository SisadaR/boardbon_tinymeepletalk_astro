เกมนี้เป็นหนึ่งในผลผลิตจากกิจกรรมของทางกลุ่ม 
Board Game Designer Thailand - BGDT ที่ผมมีโอกาสได้ลองเล่น ผลงานของคุณ Charkris Ming Ti  (Automotive Factory)
#thaidev #prototype #first_impression
 
 
ขอเริ่มที่ตัวกิจกรรมก่อน คือทางกลุ่ม BGDT เนี่ยช่วงนี้เค้าออกกิจกรรมที่จะคอยคิดหัวข้อให้นักออกแบบมาร่วมสนุกกันโดยเลือกหัวข้อขึ้นมาอย่างหนึ่งแล้วก็ไปทำกันมา (มี ตีรันฟันทอย บอร์ดเกม คาเฟ่ - Tea Run Fun Toy Board Game Cafe คอยซับเรื่องสถานที่กับรางวัลให้) ซึ่งหัวข้อรอบนี้คือ รอนเดล (Rondel) + การศึกษา (Education) เป็นระบบการเล่นเกมที่นักออกแบบ Mac Gerdts เอาไปใช้จนโด่งดังขึ้นมา (Navegador ,Imperial) ลักษณะพิเศษโดยทั่วไปของระบบนี้คือ
 
 
- วางแอคชั่นไว้ที่จะทำไว้เป็นวงกลม
.
- ผู้เล่นต้องเอาคนงานเดินไปตามวงกลมที่ว่าเพื่อทำแอคชั่น
.
- ผู้เล่นถูกจำกัดตัวเลือกให้อยู่แค่ไม่กี่ช่องด้านหน้า (บางเกมถ้าจะเดินเพิ่มต้องจ่ายทรัพยากร)
.
- การเลือกที่จะเดินข้ามแอคชั่นมีผลต่อการตัดสินใจสูงเพราะต้องรอเดินครบกลับมาอีกรอบถึงจะได้ทำแอคชั่นนี้อีกครั้ง
 
มาคิดๆแล้ว.....อย่าง Monopoly ก็น่าจะได้มั้ง.....ฮา
 
 
กลับมาที่เกมนี้ต่อ....ทางนักออกแบบก็ตีโจทย์ออกมาเป็นเกมก้มหน้าก้มตาอ่านหนังสือเรียนไปทำข้อสอบ ในแง่ไอเดียก็คือเกมเก็บ set collection ไทล์สี (แทนวิชาเรียน) ไปแลกใบรางวัล (ข้อสอบ) ทั่วไปแต่ส่วนที่ทำให้เกมนี้น่าสนใจมากแม้จะเป็นแค่งานทดลองก็คือ 'วิธีการเดินคนงาน' ซึ่งที่บอกว่ามันน่าสนใจมากคือมันน่าจะเรียกได้ว่าเป็นระบบที่ Innovative ที่น่าจะยังไม่มีคำจำกัดความสวยๆมาเรียกมัน
 
 
คืองี้ เราจะเริ่มจากเดินคนงานไป 1-3 ช่อง (ต้องทอยเต๋าเพื่อหาเลขสูงสุดที่เดินไปได้ แต่เดินไม่ถึงก็ได้) พอเดินแล้วเราก็จะต้องหยุดอ่านหนังสือโดยเลือกได้ว่าจะหยุดอ่านนานแค่ไหนยิ่งหยุดมากก็จะหยิบไทล์สีตรงจุดนั้นได้มากตาม แต่แลกกับการที่เราต้อง 'สไลด์' เลื่อนตัวไปข้างหน้าเยอะขึ้น (เปรียบเสมอใช้เวลาอ่านหนังสือ) พอหยุดตรงไหนเราก็สามารถหยิบใบทำแต้มมาเคลมได้ ทีนี้ไอ้การสไลด์ถ้าไปไกลของเราได้เยอะก็จริง แต่มันจะทำให้เรา 'จบวัน' หรือจบรอบเร็วกว่าคนอื่น ตรงนี้เราก็ต้องไป weight น้ำหนักเอาระหว่าง จำนวนไทล์, ใบทำแต้มที่เราอยากได้, จำนวนแอคชั่นทีทำได้ และ เทิร์นออเดอร์ (เพราะออร์เดอร์เรียงตามระยะทางที่เดินไป ใกล้กว่าได้เลือกก่อน) ก็วนๆกันจนวิ่งเป็นวงกลมครบสามรอบ
 
 
ถ้าพูดอีกแบบก็คือเกมนี้จะให้เรา เดิน->สไลด์->หลับ->ตื่น วนลูป
 
 
ในแง่หนึ่งเกมนี้เป็นยูโรระดับกลางเบาไม่ค่อยจะ 'รอนเดล' เท่าไร แต่ระบบเดินคนงานนี้ค่อนข้างประทับใจ ตัวเกมเองก็เป็นเกมแนวคิดแบบนวดหัวตลอดเวลาแต่ไม่รู้สึกว่าล้าหรือเบื่อ downtime ต่ำ แต่ว่าอาจจะเยอะไปซักนิดถ้าเล่นมากกว่าสามคน และส่วนตัวรู้สึกว่าจังหวะที่เราต้องมาทอยเต๋าเดินนี้มันสะดุดกับไม่ค่อยเข้ากับเกมนัก แต่คิดว่าระบบนี้น่าจะเอาไปต่อยอดได้น่าสนใจอีกเยอะอยู่
