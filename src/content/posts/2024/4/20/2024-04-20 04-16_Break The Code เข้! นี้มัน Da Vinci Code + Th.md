Break The Code

เข้! นี้มัน Da Vinci Code + The Coding + Turing Machine คิลเลอร์นิหว่า

โอเคพูดให้ดูเว่อร์ๆไป (แต่จริงๆก็คิดงั้นอยู่นะ) แต่ไอเดียเกมนี้คือมันจะมีไทล์เลข 1-9 อยู่ 2 ชุดเป็นสีขาวกับสีดำ สุ่มจั่วให้ผู้เล่นคนละ 5 ไทล์ ถ้าเราเล่นกัน 2 คนเป้าหมายคือจะแกะเลขของอีกฝ่าย แต่ถ้าเล่น 3-4 คนเราจะมาทายว่า 'ไทล์เลขที่เหลือ' อยู่ในกองกลางเป็นเลขอะไรแทน โดยการทายต้องเป๊ะทั้งสีและตำแหน่ง

---
การเรียงเลขในเกมจะมีกฎอยู่ว่าต้องเรียงจากน้อยไปมากและถ้าเลขเท่ากันต้องให้สีดำอยู่ข้างหน้า ทวิสของเกมคือการที่เราจะถามอะไรใครเราต้องเลือกหยิบจากการ์ดคำถามที่มีให้เลือก ถามแล้วจะทิ้งไปแล้วจั่วใบใหม่มาแทน 

คำถามมันก็จะแนว 'ขอผลรวม 3 ตัวแรก' , 'เลข 8 ของคุณอยู่ในตำแหน่งไหน', 'มีเลขขาวกี่ตัว' ไรงี้ ซึ่งผมชอบตรงนี้มากเพราะมันช่วยกำจัดการถามแบบถามโง่ๆเพราะนึกไม่ออก กับช่วยให้คนมีตัวช่วยในการเลือกถามไปพร้อมๆกัน เอาจริงๆคือแค่นี้ก็จีเนียสละ

---
เอาจริงๆเกมมันมีแค่นี้แหละแต่คิดว่าใครเป็นสายแนวใช้ลอจิกแกะเลขต้องชอบแน่ๆ อย่าง Da Vinci Code กับ The Coding ของอาร์มงี้ (ซึ่งจริงๆก็ไม่ได้คล้ายไร Turing Machine แค่มีบัตรคำถามเลยเนียนมาด้วยเฉยๆ)

---
เป็นอีกหนึ่งผลประกอบการณ์ช่วงสงกรานต์ พอดีช่วงนี้ทำงานใช้กรรมวุ่นมากกกเลยพึ่งว่างเอามาลง ขอบคุณ @Heavy Meeple ที่หยิบติดมาให้ลองครับ

---
Frog-o-Meter: โอเค  
  
(My) Collection Fit: ซื้อ! กล่องเล็กอธิบายง่าย เก็บไว้ใส่ collection เกมเล็ก จริงๆผมเล่นเกมแนว deduction แบบใช้ logic ไม่เก่งเท่าไร (น่าจะเพราะสมาธิชอบหลุด) เกมอย่าง Alchemist นี้คือโง่แดรก แต่ถ้าระดับประมาณเกมนี้ผมค่อนข้าง enjoy นะรู้สึกว่าไม่มากไปจนปวดหัว ไม่น้อยไปจนไม่ต้องคิด
  
What I like: ระบบการ์ดคำถามที่ช่วยให้การถามตอบลื่นไหล เกมไม่วนซ้ำกับชุดคำถามจังหวะโทนเดิมๆ
  
What I dislike: ไม่ได้มีอะไรที่ไม่ชอบเป็นพิเศษ 
  
I think this be good for... : any deduction fan
