เรื่อง: ทดลองย้าย hosting สำหรับข้อเขียน

ไม่แน่ใจว่าปกติชอบกดไปดูใน WordPress blog กันเยอะไหม แต่หลังๆผมไม่ได้เอาลงไปในนั้นด้วยเหตุผลสองอย่างคือ

* ขี้เกียจ...... เอาจริงๆก็ไม่เชิง (แต่ก็ขี้เกียจนั้นแหละ) คือมันต้องพิมพ์สองรอบเพราะ WordPress มันจัดบรรทัดคนละแบบกับ fb เวลา copy paste ไปแล้วต้องมาแก้กันรายบรรทัด พ่วง ui ใส่รูปมันกดได้น่ารำคาญมักๆ

* ก็เรื่องรูปนั้นแหละ คือ....ที่เต็มครับ เค้าให้มา 3GB ใช้หมดแล้ว เพราะเมื่อก่อนไม่ได้ resize รูปไฟล์เลยค่อนข้างใหญ่แม้จะเป็น jpeg จะให้ไปไล่ย้อน resize รูปก็คิดว่าเยอะขนาด

ตอนนี้เลยกะว่าจะเอาไปลงที่ GitHub แทน ที่ไม่มีความหวือหวาอะไรทั้งนั้นแต่มัน ฟรี ฟรี ฟรี! หน้าตามันก็จะดู nerd ประมาณหนึ่งแต่สามารถกดจิ้มชื่อไฟล์ได้เลยนะ ในแง่หนึ่งมันก็จะค้นหางานเก่าๆง่ายกว่า เพราะจริงๆที่แยกไปลงใน WordPress ก็เพราะอยากเก็บไว้ให้หาง่ายๆนั้นแหละ ไม่ได้หวังเรื่องความสวยงามอะไรขนาดนั้น 

เท่าที่ลองกับมือถือตัวเองก็คิดว่ามันอ่านง่ายอยู่นะ พอๆกับตอนใช้  WordPress 

จะมีข้อเสียหน่อยๆก็คือสถิติพวกจำนวนคนที่มาดูต่อวันว่าดูหน้าไหนบ้างจะหายไป แต่เอาจริงๆก็ไม่ได้สนใจเท่าไรอยู่แล้วก็คงไม่เป็นไรมั้ง?.....

คิดเห็นอย่างไรก็บอกกันได้ครับ (จริงๆเมื่อก่อนเอาไปลง Blockdit ด้วยแต่ก็เลิกเอาลงเพราะต้องทำในมือถือเท่านั้น interface ก็กากเหลือหลาย ใช้บนเวบก็ไม่ได้)
